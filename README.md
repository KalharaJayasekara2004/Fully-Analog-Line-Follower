# Fully Analog Line Following Robot

## 📌 Overview

This project presents the design and implementation of a fully analog line following robot developed for the EE2030 module at the University of Peradeniya.

The robot operates entirely without microcontrollers and is based on analog circuitry, logic gates, and flip-flop based control logic.

---

## ⚙️ Features

- Fully analog implementation
- No microcontroller usage
- SR flip-flop based control logic
- NAND gate logic optimization
- PCB-based hardware design
- Dash-line handling capability

---

## 🔧 Components Used

- KY-033 sensors
- L293D motor driver
- SR flip-flops
- NAND gates
- N20 DC motors
- Li-ion batteries
